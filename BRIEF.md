# Summary
A dashboard for FastForward Logistics, a mid-size freight and supply chain company, with a clean design. The VP of Operations wants a single internal dashboard she can pull up in leadership meetings to see how the business is running: shipment volume, on-time delivery rates, regional performance, and open exceptions

## Tech 
- Pure HTML, CSS, and JavaScript - no frameworkds, no build tools
- Single index.html file (CSS and JS can be separate files)
- Deploys to Vercel as a static site

## Design
- Modular card layout
- FastForward Logistics Dashbaord as a heading, with a short blurb underneath
- Include mock logo next to heading
- Dark mode by default with a light/dark toggle
- smooth hover animations on buttons and clickable cards
- Googe Font: Lato
- use green and teal in light mode, purple and blue in dark mode. Keep it nice and pretty

## Nice to have
- Footer with "Made with candy and Copilot" 
- Include gradients on charts where applicable
- Live updates for applicable stats and graphs
